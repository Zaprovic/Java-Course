Polymorphism

This is a confusing one to explain, but is basically making sure that any method inside a class is capable of identifying to which class is referring to. Let us consider again the same code from above:

public static class Country{
int population;
String name;
String language;

public String printLanguage(){
return "Here in " + this.name + " we speak " + this.language;
    }

public void printMessage(){
System.out.println("I am a citizen of the world");
   }
}

public static class France extends Country{
@Override
public void printMessage(){
System.out.println("I am a french person");
   }
}

public static void main(String[] args){
Country country1 = new Country();
Country country2 = new France();

country1.printMessage();
country2.printMessage();
}

Here in this code we have a parent class named Country and a child class named France that extends to Country. In the parent class we have a method named printMessage() and we overrode it to the child class.

When creating the instance country1 we see it is type Country; and then created another instance country2 that is type France. When the driver class is executed, we will see two different messages in the terminal, each one correspond to the class it was instantiated.
